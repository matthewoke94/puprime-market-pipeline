# Forex Market Data Pipeline

A production-grade ETL pipeline that ingests live forex market data, applies financial transformations, and loads enriched data into a PostgreSQL database. Built to demonstrate real-world data engineering skills aligned with fintech and online trading platforms.

## Project Overview

This pipeline automates the collection and processing of forex price data for currency pairs traded on platforms like PuPrime. It runs on a schedule and stores clean, analytics-ready data in the cloud.
